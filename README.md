# 🎄 圣诞祝福 H5 小程序

一个充满创意和心意的圣诞祝福互动网页，让您为不同身份的朋友送上专属的圣诞祝福！

## ✨ 功能特点

- 🎁 **7种关系场景**：恋人、朋友、家人、同事、邻居、师生、上下级
- 🎨 **精美动画效果**：Canvas粒子系统、雪花飘落、爱心烟花
- 💌 **个性化祝福**：系统预设 + 自定义祝福语
- 📱 **完美适配移动端**：支持微信分享
- 🚀 **纯前端实现**：无需服务器，静态托管即可

## 🎯 技术栈

- **前端框架**: HTML5 + CSS3 + Vanilla JavaScript
- **动画引擎**: Canvas API（粒子动画系统）
- **数据传输**: URL参数 + Base64编码
- **部署方式**: 静态托管（Vercel / Netlify / GitHub Pages）
