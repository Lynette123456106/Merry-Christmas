## 🚀 立即运行（三种方式）

### 方式1：直接打开（最简单）
双击 `index.html` 文件即可在浏览器中预览。

### 方式2：使用Python服务器（推荐）
```powershell
# 在项目目录下运行
python -m http.server 8000

# 然后访问：http://localhost:8000
```

### 方式3：使用VS Code Live Server
1. 安装 Live Server 扩展
2. 右键 `index.html` → Open with Live Server

### 4. 部署上线

#### ⭐ 推荐：Vercel（最简单）
1. 访问 https://vercel.com
2. 点击 "New Project"
3. 导入你的 GitHub 仓库
4. 一键部署
5. 获得免费的 HTTPS 域名

#### GitHub Pages
1. GitHub仓库 → Settings → Pages
2. Source 选择 `main` 分支
3. 保存后等待几分钟
4. 访问 `https://你的用户名.github.io/Merry-Christmas`

#### Netlify
1. 访问 https://netlify.com
2. 拖拽项目文件夹到网页
3. 立即部署

## ⚠️ 常见问题

### Q1: 二维码不显示？
A: 需要网络连接加载CDN资源（qrcode.js）

### Q2: 音乐无法播放？
A: 
1. 确保 `assets/audio/jingle-bells.mp3` 文件存在
2. 浏览器需要用户交互后才能播放音频

### Q3: 展示页面空白？
A: 检查URL参数是否完整，尝试重新生成链接

### Q4: 样式错乱？
A: 按 Ctrl+F5 强制刷新浏览器缓存
